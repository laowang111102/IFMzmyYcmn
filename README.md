# Python航空数据分析系统 python566

## 项目概述

本项目是基于Python的航空数据分析系统，系统包括源码及6千字的开发文档。系统采用Flask作为后端框架，使用MySQL存储数据，并通过HTML、CSS、JavaScript以及Layui进行前端展示。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript, Layui

## 功能模块

- **用户系统**
  - 登录注册
  - 个人信息修改
- **航班数据管理**
  - 分页查询
  - 用户管理
- **数据分析**
  - 不同航空公司航班数据
  - 不同出发城市航班数据
  - 不同航班机型数据
  - 平均准点率
  - 平均误点率
  - 不同出发省份航班分析
  - 到达多个城市分析

## 系统角色

- 普通用户：可查看和分析航班数据，管理个人信息
- 管理员：拥有用户管理权限，以及对航班数据的全面管理

## 运行环境

- Python环境
- Flask框架
- MySQL数据库
- 支持HTML、CSS、JavaScript的Web浏览器

## 部署步骤

1. 安装Python环境
2. 安装Flask框架
3. 配置MySQL数据库
4. 运行系统

具体步骤见开发文档。
